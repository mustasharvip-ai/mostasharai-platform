# Mostasharai Platform

A professional consulting platform (منصة مستشاري) that connects users with specialized consultants in business, law, technology, and finance.

## Tech Stack

- **Runtime:** Node.js 20
- **Framework:** Express.js
- **Frontend:** Vanilla HTML/CSS (Arabic RTL layout)
- **Package Manager:** npm

## Project Structure

```
mostasharai-platform/
├── server.js          # Express server (port 5000, host 0.0.0.0)
├── package.json       # npm configuration
├── public/
│   └── index.html     # Main landing page (Arabic RTL)
└── .gitignore
```

## Running the App

```bash
npm run dev
# or
node server.js
```

The app runs on port 5000 at `http://0.0.0.0:5000`.

## Workflow

- **Start application**: `node server.js` on port 5000 (webview)
